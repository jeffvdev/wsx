# WorkSpaceX - WSX
## Run It / Forget It!
WorkspaceX is a cli tool that wraps your terminal session. All commands sent to **wsx** will be executed in the backgound, allowing you to keep running more commands witout having to wait for the previouns to finish.

> ***This tool is not in a working state*** *Initial work will be minimal and tool will evolve through the [Issue Tracker](https://github.com/jeffvdev/wsx/issues)*
